

# 🏥 Smart Health & Medical History with Wellness Analytics System (SHM-WAS)

## 📌 Overview

The **Smart Health and Medical History with Wellness Analytics System (SHM-WAS)** is a Python-based application designed to efficiently manage and analyze patient health data.
It provides a digital platform to store patient details and perform basic wellness analysis such as BMI calculation.

---

## 🎯 Aim

To develop a Python-based application that:

* Manages patient health records efficiently
* Provides basic wellness analysis
* Uses object-oriented programming, file handling, and data structures

---

## ✨ Features

* Add and store patient details (name, age, weight, height)
* Maintain medical history records
* BMI calculation and health categorization
* Search patient records easily
* File handling for permanent data storage
* Menu-driven user-friendly interface
* Exception handling for error-free execution

---

## 🧠 Concepts Used

* Object-Oriented Programming (Classes, Objects, Inheritance)
* File Handling
* Exception Handling
* Data Structures (Lists, Sets)
* Control Structures (Loops, Conditionals)
* String Operations

---

## ⚙️ System Requirements

### 💻 Software Requirements

* Python 3.x
* Operating System: Windows 10 or above

### 🖥️ Hardware Requirements

* RAM: Minimum 4 GB
* Processor: Intel i3 or equivalent

---

## 📊 Working (Algorithm)

1. Start the program
2. Initialize empty patient list
3. Display menu options
4. Take user input
5. Perform operations:

   * Add patient
   * Display records
   * Search patient
   * Save to file
6. Repeat until user exits
7. End program
